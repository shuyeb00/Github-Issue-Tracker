1. What is the difference between var, let, and const?

   var is old way to declare variable. It has function scope and can be changed.
   let is newer. It has block scope and value can be changed.
   const is also block scope but value cannot be changed after declaring.

2. What is the spread operator (...)?

   Spread operator (...) is used to copy or combine arrays or objects.
   Example: [... [1,2], ... [3,4]] gives [1,2,3,4].

3. What is the difference between map(), filter(), and forEach()?

   map() goes through array and changes each item and returns a new array.
   filter() goes through array and returns only items that match a condition.
   forEach() just loops through the array and does something but does not return a new array.

4. What is an arrow function?

   Arrow function is a short way to write a function in JavaScript.
   Example: const add = (a,b) => a+b

5. What are template literals?

   Template literals are strings written using backticks ` ` and we can insert variables inside using ${}.
   Example:
   `let name = "Rahim
   Hello ${name}`
